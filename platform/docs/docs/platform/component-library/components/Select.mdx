# Select

## Import

```javascript
import { Select } from '@ohif/ui';
```

## Basic usage

```jsx live
<div className="flex flex-col flex-1 p-4 items-center">
  <div className="w-56">
    <Select
      options={[
        { value: 'ONE', label: 'ONE' },
        { value: 'TWO', label: 'TWO' },
        { value: 'THREE', label: 'THREE' },
        { value: 'FOUR', label: 'FOUR' },
      ]}
    />
  </div>
</div>
```
